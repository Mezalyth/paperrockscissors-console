Rock Paper Scissors

This game is played in the console.

Right Click > Inspect > Console

Pseudocode

  1. Get R, P, or S input from user.
  2. Get R, P, or S input from machine (opponent).
  3. Define R > S
  4. Define S > P
  5. Define P > R
  6. Whichever wins, announce for round!
  7. Define equal inputs as "tie"
  8. Do this for 5 rounds
  9. If 3 victories, win!
  10. If 3 losses, lose!
  11. If 3 ties, tie!
  12. If results announced, reset counter to zero
